Animal Colony Management System

Description

A collaboration with medical researchers San Francisco.

Built with React for the frontend, Node.js for the backend, and Google Firestore for the database.

This application provides a unified solution in providing all information for a single animal or a collection of animals for researchers.

Users are able to centrally manage all animal colony related information with the benefits of visual feedback interactions and a responsive user interface.

Other features include the ability to import colonies, share colonies, modify colonies, as well as uploading notes and photos.

The colony data is stored in a No-SQL database.

Skills Used

  • JavaScript
  • React
  • Node.js
  • Google Firestore
